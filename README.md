Jeu de Démineur pour Nintendo GameBoy Advance

Ce projet est un jeu vidéo programmé en langage C et à l'aide du kit de développement HAM.

Fonctionalitées :
- Grille générée aléatoirement,
- 5 niveaux de difficultée disponibles,
- choix du nombre de mines,
- possiblilitée de sauvegarder et charger une partie dans la mémoire,
- utilisation d'un mot de passe pour regénérer une même grille.

Tous les fichiers se situent dans la même dossier à cause du compileur mais ils se déclinent en trois types :
Fichiers sources C, Fichiers d'en-têtes H, et Images/Textures RAW.C.

Projet non achevé.
Testé seulement sur émulateur.
